# Guia breu de continguts

## Actualitat

Utilitza:

- `noticia` per a informació breu;
- `cronica` per explicar una activitat realitzada;
- `article` per a textos d'anàlisi;
- `comunicat` per a posicionaments institucionals.

El resum ha de poder llegir-se de manera independent i no hauria de superar dues frases.

## Agenda

Cada activitat ha d'incloure data, hora, lloc, organització i una descripció pràctica. Quan existeixi un arxiu `.ics`, guarda'l a `public/calendar/`.

## Campanyes

Una campanya ha d'explicar:

1. el problema;
2. els objectius;
3. les línies d'acció;
4. com pot participar-hi una persona;
5. els recursos o enllaços relacionats.

## Imatges

- fotografies documentals i cartells propis;
- text alternatiu descriptiu;
- crèdit o autoria quan sigui necessari;
- permís de publicació per a les persones identificables;
- format WebP per a ús general.

## Estil

- català estàndard;
- frases directes;
- context suficient per a persones que no coneixen BAS;
- evitar llenguatge comercial;
- diferenciar fets, interpretacions i propostes;
- enllaçar les fonts originals quan sigui rellevant.
